# Salama Shop - Page d'avis clients

Cette page permet aux clients de Salama Shop de donner leur avis en scannant un QR code.

## Fonctionnalités

- ✅ Interface responsive et moderne
- ✅ 3 niveaux de satisfaction (😊 Satisfait, 😐 Moyen, 😞 Déçu)
- ✅ Commentaire optionnel
- ✅ Récupération automatique des infos depuis l'URL (code, ticket, client)
- ✅ Messages de confirmation

## Utilisation

Les QR codes générés par l'application pointent vers :
`https://TON-PSEUDO.github.io/salama-shop-avis/?code=XXXX&type=ticket&nom=Client&tel=771234567&ticket=2026/03/09-vente42`

## Installation

1. Cloner ce dépôt
2. Modifier l'URL du serveur dans `index.html` (ligne ~200)
3. Pusher sur GitHub
4. Activer GitHub Pages
